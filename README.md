# Simple Slot Game

## Installation

Just open the project in Cocos Creator program.

## Architecture

- Cocos Creator version 3.8.8
- Code split across 3 main regions: animation, UI, win
- Animations are executed by standard UI positioning
- Assets are taken from: https://github.com/gamecocos/gameSlots
- Sound is added to the game
- Game was created with the help from AI in following areas:
  - symbol animations
  - betlines creation
  - bug fixing

## Notes

- Game has a fixed size - you need to adjust resolution of your browser to play the game
  - Game <ins>is not supported on landscape mode on mobile</ins> at the time
- This game is skill-dependant. It <ins>is not standard random</ins> slot game.
